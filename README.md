# Twin-Analysis
Twin analysis code built on MTEX and Matlabs graph toolbox

## Summary
The objective of this project is to recreate and improve on the descriptions of twin analysis presented in [graph theory](https://link.springer.com/article/10.1007/s40192-018-0106-y) and [hierarchy](https://onlinelibrary.wiley.com/doi/full/10.1111/j.1365-2818.2009.03343.x) based twin analysis methodologies using an open source toolset.
